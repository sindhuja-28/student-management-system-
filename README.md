# student-management-system-
🎓 Student Management System

A web-based Student Management System built using Java, JSP, Servlets, and Maven. This application allows efficient management of student data, courses, and user authentication for both students and staff.

🚀 Features


👨‍🎓 Student Registration & Login

👩‍🏫 Staff Login & Dashboard

📚 Course Management (Add / View / Edit)

📝 Student Profile Management

🔐 Password Recovery System

📊 Dashboard for Students & Staff

📂 File handling (certificates, photos)

🛠️ Tech Stack


Backend: Java (Servlets, JDBC)

Frontend: JSP, HTML, CSS

Build Tool: Maven

Server: Apache Tomcat

Database: MySQL

Clone the repository

Import project into Eclipse / IntelliJ

Configure Apache Tomcat server

Connect MySQL database

Run the project on server
